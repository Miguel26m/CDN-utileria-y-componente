# Libreria utileria y componentes

## Implementación

Para implementar las librerias solo debes descargarlas y llamar las funcioes en tu archivo .html

## Libreria utileria.js

Es una libreria vanila, basicamente no usa framework como react, esto hace que al implementarlas tu pagina web no se vuelva lenta.

## Componente

El componente que viene en el archivo componente.js es un carrusel de fotos, este carrusel cuenta con  una foto, un titulo y una descripción, dicho sea de paso es vanila, osea tampoco se usaron framework como react, 
el componente se ppuede usar multiples veces en una pantalla, de esta forma podrias implementarlo en ciertas paginas web, como las de turismo.
